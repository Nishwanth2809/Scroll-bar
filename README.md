# Scroll-Driven Hero Section Animation

This project recreates a premium scroll-driven hero interaction using Next.js, GSAP, HTML, CSS, and JavaScript.

## Tech stack
- Next.js
- Tailwind CSS
- GSAP (ScrollTrigger)

## Local run
```bash
npm install
npm run dev
```

## Build static export
```bash
npm run build
```

## Deployment
This repository includes a GitHub Actions workflow at `.github/workflows/deploy.yml`.

After pushing to `main` and enabling GitHub Pages (Source: GitHub Actions), the site will be available at:

`https://<your-github-username>.github.io/<your-repo-name>/`

## Submission links format
- Live webpage: `https://<your-github-username>.github.io/<your-repo-name>/`
